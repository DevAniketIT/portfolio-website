# Job Application System

A comprehensive system to manage your job applications and freelance proposals with your complete portfolio.

## 🚀 Features

- **Automated application tracking** - Never lose track of applications
- **Template generation** - Create personalized emails and proposals
- **Portfolio integration** - Your live projects showcased in every application
- **Follow-up management** - Automatic reminders for follow-ups
- **Statistics dashboard** - Track your success rates

- **Live App:** https://aniket-job-tracker.streamlit.app
- **API Docs:** https://job-tracker-api-aniket.onrender.com/docs  
- **GitHub:** https://github.com/aniketkumar7/aniket-portfolio
- **Price Monitor:** https://github.com/aniketkumar7/aniket-portfolio/tree/main/price-monitor

## 🛠 Setup (Windows)

### 1. Navigate to the job-applications directory:
```bash
cd job-applications
```

### 2. Run the application manager:
```bash
py application-manager.py
```

### 3. Install required packages if needed:
```bash
py -m pip install requests
```

## 📋 Quick Start

### Start Your First Day:
1. **Test your portfolio links** (make sure they're all working)
2. **Run the application manager:** `py application-manager.py`
3. **Generate your first application** (option 4 in the menu)
4. **Apply to 5 high-priority startups** (see target-companies.md)
5. **Send 5-10 freelance proposals** (see freelance-proposals.md)

### Daily Routine:
- **Morning (9-12 PM):** Apply to 3-5 jobs, 5-10 freelance projects
- **Afternoon (2-4 PM):** Follow-ups and research
- **Evening (6-7 PM):** Update tracker and plan tomorrow

## 🆘 Troubleshooting

### Portfolio Links Not Working:
- Check Streamlit Cloud dashboard
- Verify Render deployment status
- Contact hosting support if needed

### Application Manager Errors:
- Make sure you're in the job-applications directory
- Use `py` command instead of `python` on Windows
- Install missing packages: `py -m pip install requests`

### No Responses After 1 Week:
- A/B test different email subjects
- Focus more on smaller companies
- Increase freelance proposal volume
- Enhance portfolio with additional features
